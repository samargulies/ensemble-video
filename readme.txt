Easily embed ensemble videos in your site

== Description ==

Easily embed ensemble videos in your site with the `[ensemblevideo]` shortcode

== Installation ==

1. Install the Ensemble Video plugin in `wp-content/plugins/`
2. Activate at Dashboard > Plugins
